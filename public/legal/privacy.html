Politique de Confidentialité

Cette politique explique comment Serrurerie Joseph collecte, utilise et protège les informations fournies par les utilisateurs.

1. Données collectées

Le GPT Serrurerie Joseph peut collecter les informations suivantes lorsque l’utilisateur les communique volontairement :

  • Nom
  • Numéro de téléphone
  • Adresse ou localisation
  • Description du problème de serrurerie

2. Utilisation des données

Les données fournies servent uniquement à :

  • Fournir une assistance liée aux services de serrurerie
  • Répondre aux demandes d’informations ou de conseils

3. Partage des informations

Serrurerie Joseph ne partage aucune donnée personnelle avec des tiers. Aucune donnée n’est stockée par le GPT.

4. Sécurité

Serrurerie Joseph prend toutes les mesures raisonnables pour protéger les données transmises par les utilisateurs.
